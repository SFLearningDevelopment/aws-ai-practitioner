Core AI / Machine Learning Concepts

Domain 1 · AIF-C01 · 21 Key Terms
Domain 1
Term Definition
Artificial Intelligence (AI) Systems that simulate human intelligence to perform tasks like reasoning, learning, and problem solving.
Machine Learning (ML) A subset of AI where models learn patterns from data to make predictions — without being explicitly programmed.
⟶ Data → Train → Predict
Deep Learning (DL) ML using multi-layered neural networks. Excels at images, text, and audio. Requires large datasets and compute.
Supervised Learning Model trained on labeled data (input → known output). Used for classification & regression.
Example: spam detection, price prediction
Unsupervised Learning Model finds patterns in unlabeled data. Used for clustering, anomaly detection, dimensionality reduction.
Example: customer segmentation
Reinforcement Learning Agent learns by trial-and-error, receiving rewards/penalties for actions in an environment.
Example: game-playing bots, robotics
Training Data Dataset used to teach the model. Quality & quantity directly affect model performance.
Inference Using a trained model to make predictions on new, unseen data. (Training ≠ Inference)
Overfitting Model memorizes training data but fails on new data.
Fix: more data, regularization, dropout
Underfitting Model is too simple to capture patterns.
Fix: more complex model or more features
Feature Engineering Selecting, transforming, or creating input variables to improve model accuracy.
Model Evaluation Metrics Accuracy — % correct
Precision — of predicted positives, how many are real
Recall — of real positives, how many did we catch
F1 Score — harmonic mean of precision & recall
AUC-ROC — overall discriminative power
Confusion Matrix Table showing TP, TN, FP, FN. Used to evaluate classification model performance.
Hyperparameter Configuration set before training (e.g. learning rate, number of layers). Not learned from data.
Cross-validation Technique to assess model generalization by splitting data into folds and rotating train/test sets.
Neural Network A computing model inspired by the human brain, made of layers of interconnected nodes (neurons). The foundation of deep learning.
Transfer Learning Reusing a model pre-trained on one task as the starting point for a new, related task. Saves time and data.
Example: using a pre-trained image model and fine-tuning it for medical scans
Classification Predicts which category an input belongs to. Output is a discrete label.
Example: spam / not spam, cat / dog / bird
Regression Predicts a continuous numerical value as output.
Example: predicting house price, temperature, stock value
Labeled vs Unlabeled Data Labeled — data with known output answers; used in supervised learning.
Unlabeled — raw data with no answers attached; used in unsupervised learning.
